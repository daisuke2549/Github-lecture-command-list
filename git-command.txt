○コマンドリスト

git init  =   リポジトリを新規に作成するコマンド。手元のPCでローカルリポジトリーを作る際に使用


git add   =  コミットしたいファイルをステージに追加する     

git add . で全てのファイルを選択。git add ファイル名で対象のファイルのみを選択可能



git commit(コミット) =   作業ディレクトリーで作業した内容をローカルリポジトリーに反映すること。


git status  = 現在のgitの状態を確認


git branch(ブランチ) =  ブランチの一覧を表示するコマンド。　git branch (新しいブランチ名で新ブランチの作成が可能)


今日現在、新たに作成するすべてのGitHubリポジトリは、'master'ではなく'main'ブランチがデフォルトブランチとなっている


git remote add  = リモートリポジトリと紐付けるために使うコマンド

git push =  ローカルリポジトリーの内容をリモートリポジトリーに送信するコマンド


git pull = リモートリポジトリの内容を取得して、現在のブランチに取り込む（「fetch」と「merge」を行う）

git clone  = リモートリポジトリーのクローンを作成する。

git diff = 差分を比較するコマンド

git checkout  = ブランチの切り替えのために使う　git checkout(ブランチ名)

git log = コミットログを表示
